Video Downloader (Python)
A simple Python based video downloader that allows users to download videos in different quality options using yt-dlp.
⚠️ This project is created for educational purposes only. Please respect the Terms of Service of YouTube, spotify and only download content you have permission to use.
🚀 Features
Download videos using a YouTube URL
Choose video quality (720p, 480p, or Best Available)
Automatically merges video and audio
Saves video with original title
Simple and beginner-friendly code
Can be extended to GUI version
🛠 Technologies Used
Python 3.x
yt-dlp library
